# 🏢 Mi Técnico PC - Sistema de Gestión Logística

Sistema web para la optimización del control logístico, inteligencia de compras y trazabilidad de inventarios en Mi Técnico PC.

---

## 📌 Descripción del Proyecto

**Mi Técnico PC Logístico** es un aplicativo web desarrollado a medida para resolver los problemas de control de inventario, compras duplicadas y pérdida de herramientas en la empresa Mi Técnico PC, una MYPE dedicada al soporte técnico informático.

El sistema permite:
- ✅ Gestionar el catálogo de productos (repuestos, herramientas, licencias)
- ✅ Registrar compras con historial de costos
- ✅ Controlar despachos vinculados a órdenes de servicio
- ✅ Gestionar préstamos de herramientas con trazabilidad
- ✅ Visualizar dashboards con KPIs en tiempo real

---

## 🛠️ Tecnologías Utilizadas

| Capa | Tecnología |
|------|------------|
| **Backend** | Google Apps Script (GAS) |
| **Base de datos** | PostgreSQL (Supabase) |
| **Frontend** | HTML5, CSS3, JavaScript (Vanilla JS) |
| **Framework UI** | Bootstrap 5 |
| **Arquitectura** | Serverless / SPA (Single Page Application) |
